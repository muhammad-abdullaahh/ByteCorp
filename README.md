# ByteCorp Internship

A collection of tasks, exercises, and mini-projects completed during my software engineering internship at ByteCorp. This repo documents my progress across backend development (FastAPI), database design (PostgreSQL), and Python fundamentals.

## 📁 Repository Structure

```
ByteCorp/
├── Fast api/                          # FastAPI learning track & CRUD exercises
├── To do list (Postgree x python)/    # CLI To-Do app with PostgreSQL integration
├── ERD.png                            # Entity-Relationship diagram (schema design)
└── Schema.sql                         # PostgreSQL schema implementation
```

## 🚀 Contents

### `Fast api/`
Learning track covering FastAPI fundamentals, progressing from core concepts to a working CRUD project (student management system) — routes, request/response models, and basic API design.

### `To do list (Postgree x python)/`
A command-line Task Manager built in Python, connecting to a local PostgreSQL database via `psycopg2` for persistent storage — covering DB connectivity, CRUD operations, and basic schema interaction from application code.

### `ERD.png` & `Schema.sql`
Relational database design exercise: an ER diagram and corresponding PostgreSQL schema, covering entity relationships, constraints, and business rules enforced at the database level.

## 🛠️ Tech Stack
- **Languages:** Python, JavaScript, HTML, CSS
- **Backend:** FastAPI
- **Database:** PostgreSQL

## 📌 Notes
This repo is updated as new tasks are assigned during the internship. Each folder is self-contained with its own code for the corresponding task.
